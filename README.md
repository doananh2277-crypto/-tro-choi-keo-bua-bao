## 🚀 How to Play

This game is simple and runs right in your terminal (or Google Colab)!

1.  **Run the script:** Click the "Run" button in Google Colab.
2.  **Make your choice:** When prompted, type one of the following:
    * `rock` (or `kéo` - the game understands both!)
    * `paper` (or `búa`)
    * `scissors` (or `bao`)
3.  **See the result:** The computer will make a random choice, and the winner will be announced instantly.
4.  **Have fun!** 🎉

> **Pro Tip:** Thanks to Python's `.lower()` function, you don't need to worry about capitalization. Type it however you like! 😉
